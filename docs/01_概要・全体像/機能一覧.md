# ⚙️ 機能一覧 - Timekeeper

---

## 認証・アカウント管理

- **F01**: ライセンス購入機能（¥10,000）
- **F02**: デバイスID 発行・管理

## 制限管理

- **F03**: アプリ使用時間監視（AccessibilityService）
- **F04**: 使用時間記録・残時間計算 (Room)
- **F05**: 制限超過検知・ロック表示

## 課金・アンロック

- **F06**: デイパス購入機能
- **F07**: Stripe 決済連携
- **F08**: 購入履歴管理 (Firestore)

## 設定・ユーティリティ

- **F09**: 目標時間設定
- **F10**: エラー案内画面
- **F11**: SharedPreferences 管理
